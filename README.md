# QueueNova - Smart Queue Management System

## 📌 Project Overview

QueueNova is a Smart Queue Management System developed to reduce waiting time and improve customer service management. The system allows users to register, login, and book queue tokens digitally.

## 🚀 Features

- User Registration
- User Login System
- Digital Token Booking
- Queue Management
- Token Status Tracking
- Dashboard Interface
- SQLite Database Integration

## 🛠️ Technologies Used

- Python
- Flask Framework
- HTML5
- CSS3
- JavaScript
- SQLite Database
